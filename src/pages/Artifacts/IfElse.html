🎨 Tipare de Gândire în Cod

1. Validarea (Portar la club)

# Gândirea: "Poate intra doar dacă..." if varsta >= 18 and are_bilet and not e_plin: print("Bun venit!") else: print("Nu poți intra")

2. Clasificarea (Sortator automat)

# Gândirea: "În ce categorie se încadrează?" if scor >= 90: nivel = "Expert" elif scor >= 70: nivel = "Avansat" elif scor >= 50: nivel = "Intermediar" else: nivel = "Începător"

3. Fallback (Plan B, C, D...)

# Gândirea: "Încearcă asta, dacă nu merge, încearcă altceva" if metoda_plata == "card" and card_valid: procesare_card() elif metoda_plata == "paypal" and paypal_conectat: procesare_paypal() elif are_credit_magazin: foloseste_credit() else: print("Adaugă metodă de plată")

🧩 Exerciții Mentale

Secretul: Înainte să scrii cod, gândește-te cum ai explica decizia unui prieten.

if e_weekend and treaba_terminata: print("Film time! 🎬") else: print("La muncă! 💼")

if total > 100 or client_premium or prima_comanda: transport = 0 print("Transport gratuit! 🚚") else: transport = 15
